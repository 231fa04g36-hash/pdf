# PDF Chatbot Frontend

A production-ready React + Vite application for chatting with PDF files using Retrieval-Augmented Generation (RAG).

## Tech Stack
- **Framework**: React 18 + Vite
- **Styling**: Plain CSS with CSS Modules
- **State Management**: Zustand (Chat/Conversations) & React Context (Auth)
- **HTTP client**: Axios
- **Notifications**: React Hot Toast
- **File Upload**: React Dropzone
- **Routing**: React Router DOM

## Getting Started

### Prerequisites
- Node.js (v18.x or v20.x or v24.x)
- npm (v9.x or later)

### Installation
1. Install dependencies:
   ```bash
   npm install
   ```

2. Create environment configuration:
   ```bash
   cp .env.example .env
   ```

### Running Locally
To launch the Vite development server:
   ```bash
   npm run dev
   ```

### Code Formatting and Quality
This project uses ESLint and Prettier for quality assurance.
To run the linter:
   ```bash
   npm run lint
   ```
